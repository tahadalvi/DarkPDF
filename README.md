# DarkPDF (LightPDF-style Toolkit)

This project sets up a full-stack application with a Python/FastAPI backend and a Next.js frontend for performing various PDF manipulations.

## Prerequisites

- Docker and Docker Compose

## Quick Start

```bash
docker compose up --build
```

API will be available on http://localhost:8000 (health: /health) and the web UI on http://localhost:3000.
